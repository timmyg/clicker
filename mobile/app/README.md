### mobile app

##### native releases

1. bump `package.json` version number
2. native:
   - android: `build.gradle` update **versionCode** and **versionName**
   - iOS: click on App in Xcode > General > update **Version** and **Build**
3. `npm run build:android:prod && npm run build:ios:prod`
